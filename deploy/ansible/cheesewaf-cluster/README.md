# CheeseWAF Cluster Ansible Package

This directory documents the package generated by `cheesewaf cluster` deployment tooling.

The generated package contains:

- `inventory.ini`
- `group_vars/all.yml`
- `playbook.yml`
- `roles/cheesewaf/tasks/main.yml`
- `roles/cheesewaf/templates/cheesewaf.yaml.j2`
- `README.md`

The package must not contain raw SSH passwords, private keys, API tokens, or one-time join token values. Operators should provide SSH access through Ansible inventory variables, SSH agent forwarding, a CI secret store, or the temporary SSH deployment runner in the CheeseWAF console.

Two WAF nodes are treated as load balancing, not full high availability. Minimum HA requires two WAF nodes plus one monitor node. Multi-node HA requires three or more WAF nodes and majority confirmation.
